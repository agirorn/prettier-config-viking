# prettier-config-viking

> [Prettier][prettier-url] config for Vikings

## Installation

```bash
yarn add prettier-config-viking --dev
echo '"prettier-config-viking"' > .prettierrc.json
```

## Usage

### Check the format

```bash
prettire --check src
```

### Reformat code

```bash
prettire --write src
```


[prettier-url]: https://prettier.io/
